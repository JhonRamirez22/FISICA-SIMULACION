# Simulación de cargas eléctricas 2D (Monte Carlo)

## Ejecutar

1. Instalar dependencias:
   - `pip install -r requirements.txt`
2. Correr simulación:
   - `python main.py --iterations 100000 --frame-interval 500`

## Salidas

En `outputs/` se generan:
- `frames/` (PNG por iteración guardada)
- `simulation.mp4`
- `energy_curve.png`
- `initial_vs_final.png`
- `potential_map.png`
- `electric_field.png`
- `energy_trace.csv`
- `final_state.csv`
